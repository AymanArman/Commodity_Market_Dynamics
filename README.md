The Commodity Market Dynamics app is designed for portfolio managers or risk managers who are new to the commodities space and can benefit from an interactive primer on topics across crude markets, refined product markets, and the natural gas market. 

The app can be used via a docker container. If you have docker desktop downloaded on your computer - run the following command in your terminal --> docker run -p 3838:3838 aymanarman/commodity-market-dynamics

If you want to view the app directly, the code is available in the "final_app" folder in the repository. 

Quick App Overview:

1. Forward Curves: Explore the shape, structure, and principal components of commodity forward curves across
  tenors.
2. Volatility: Return distributions, tenor correlations, and rolling realised volatility across geopolitical
  regimes.
3. Market Dynamics: Supply, demand, and spread drivers across crude, refined products, and natural gas markets.
4. Cross-Market Relationships: Rolling correlations and impulse responses between commodities across volatility
  regimes.
5. Hedging Analytics: Swap pricing, roll yield, options pricing, and dynamic hedge ratios across instruments and
  tenors.
